# Product Manager for Google Sheets

A Google Apps Script based product management system.

## Features

- Product Master
- Category Master
- Sub Category Master
- Auto Product ID
- Auto SKU
- Image Preview
- Dashboard
- Reports
- Import / Export
- Search
- Duplicate Checker

## Installation

1. Create a new Google Spreadsheet.
2. Open Extensions → Apps Script.
3. Create the `.gs` files from the `src` folder.
4. Copy the code into each file.
5. Run `installSystem()` once.
6. Refresh the spreadsheet.

## Requirements

Google Workspace or personal Google account.
